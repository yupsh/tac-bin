# yup-tac

```
NAME:
   tac - concatenate and print files in reverse

USAGE:
   tac [OPTIONS] [FILE...]

      Write each FILE to standard output, last line first.
      With no FILE, or when FILE is -, read standard input.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --separator value, -s value  use STRING as the separator instead of newline
   --before, -b                 attach the separator before instead of after (default: false)
   --regex, -r                  interpret the separator as a regular expression (default: false)
   --help, -h                   show help
```
